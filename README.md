## Useful commands

```bash
# run REST application
$ npm run start

# unit tests
$ npm run test

# show pending database migrations
$ npm run db migration:show

# run all pending database migrations (update the database structure)
$ npm run db migration:run

# use the Nest CLI
$ npx nest --help
```

See the configured [npm scripts](/package.json) for a list of all lifecycle commands.
